## 🌟 소개

학원 수강료 환불액을 **학원법**에 따라 정확하게 계산하는 웹 기반 계산기입니다. 복잡한 환불 규정을 자동으로 적용하여 투명하고 정확한 환불액을 산출합니다.

### 법적 근거
- **학원의 설립·운영 및 과외교습에 관한 법률 시행령 제18조**
- **별표 4: 학습비 등의 반환 기준**

## ✨ 주요 기능

### 1. 자동 월별 수강료 계산
- 수강 기간 입력 시 월별 수업일수 자동 계산
- 주말 제외 옵션 (토요일, 일요일 개별 선택)
- 공휴일 자동 제외 (2026년 한국 공휴일 기본 포함)
- 수업일수에 비례한 정확한 월별 수강료 배분

### 2. 수업 제외 기간 설정
- 개학, 방학, 휴강 등으로 수업이 없는 기간 설정
- 시작일과 종료일 범위 지정
- 제외 기간은 수업일수 및 수강료 계산에서 자동 제외

### 3. 정확한 환불액 계산
- **교습 시작 전**: 전액 환불
- **교습 1/3 미만 경과**: (수강료 - 일할 계산 금액) × 2/3
- **교습 1/3~1/2 경과**: (수강료 - 일할 계산 금액) × 1/2
- **교습 1/2 이상 경과**: 환불 불가

### 4. 사용자 친화적 인터페이스
- 반응형 디자인 (모바일/태블릿/데스크톱 최적화)
- 자동 날짜 포맷팅 (YYYY-MM-DD)
- 숫자 키보드 지원 (모바일)
- 상세한 계산 과정 표시

### 5. 편리한 초기화 기능
- 전체 초기화 버튼으로 모든 입력 내용 한 번에 삭제
- 새로운 계산 시작 시 편리
- 확인 메시지로 실수 방지

## 📖 사용 방법

### Step 1: 기본 정보 입력
1. **납부한 수강료** 입력
2. **수강 시작일** 및 **수강 종료일** 입력

### Step 2: 제외 조건 설정 (선택사항)
- **주말 제외**: 토요일, 일요일 제외 여부 선택
- **공휴일 제외**: 공휴일 날짜 입력 (쉼표로 구분)
- **수업 제외 기간**: 개학, 방학 등으로 수업이 없는 기간 설정

### Step 3: 월별 수업일수 계산
- "📅 월별 수업일수 계산하기" 버튼 클릭
- 각 월별 수업일수와 수강료 확인
- 각 월별 완료한 수업일수 입력

### Step 4: 환불액 계산
- **환불 고지일** (수강 포기일) 입력
- "환불액 계산하기" 버튼 클릭
- 환불 가능 금액 및 상세 내역 확인

## 📐 환불 규정

### 학원법 시행령 제18조 별표 4

| 교습 진행 상황 | 환불 비율 |
|--------------|----------|
| 교습 시작 전 | 100% (전액) |
| 총 교습의 1/3 미만 경과 | (수강료 - 일할 계산 금액) × 2/3 |
| 총 교습의 1/3 이상 ~ 1/2 미만 경과 | (수강료 - 일할 계산 금액) × 1/2 |
| 총 교습의 1/2 이상 경과 | 환불 불가 |

### 일할 계산 방식
일할 계산 금액 = (해당 월 수강료 ÷ 해당 월 총 일수) × 교습한 일수
환불 대상 금액 = 해당 월 수강료 - 일할 계산 금액
최종 환불액 = 환불 대상 금액 × 법정 비율


### 계산 예시

**조건:**
- 총 수강료: 1,000,000원
- 수강 기간: 2026-01-05 ~ 2026-02-28 (총 40일, 주말 제외)
- 1월: 20일, 2월: 20일
- 완료: 1월 5일 (5/20 = 25%, 1/3 미만)

**계산:**
1. 1월 수강료: 500,000원
2. 일할 계산: 500,000 ÷ 20 × 5 = 125,000원
3. 환불 대상: 500,000 - 125,000 = 375,000원
4. 법정 비율(2/3): 375,000 × 2/3 = 250,000원
5. 2월 전액: 500,000원
6. **총 환불액: 750,000원**
